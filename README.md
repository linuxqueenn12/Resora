# Resora
تجميعة كورسات في مختلف مجالات  علوم الحاسوب  Collection of Computer Science Courses

Resora is a project that provides a range of courses in Computer Science majors . for now only in each field 4 courses , our vision is to add popular books for each field and add a list of courses and studying plan for each field 


ريسورا مشروعٌ يُقدّم مجموعةً من الدورات في تخصصات علوم الحاسوب. حاليًا، يضمّ المشروع أربع دورات فقط في كل مجال، ونسعى إلى إضافة كتبٍ شائعةٍ لكل مجال، بالإضافة إلى قائمةٍ بالدورات وخططٍ دراسيةٍ لكل مجال يشرفني تجربتكم  💗
